# Gym Management System

A web-based Gym Management System developed using PHP, MySQL, Bootstrap, HTML, CSS, and JavaScript. This application helps gym administrators efficiently manage members, membership details, and gym records through a simple and user-friendly interface.

## Overview

The Gym Management System is designed to automate and simplify the daily operations of a gym. It provides an organized platform for managing member information, membership plans, and administrative tasks, reducing manual work and improving efficiency.

## Features

* Member Registration and Management
* Add, Update, Delete, and View Member Records
* Membership Plan Management
* Responsive User Interface using Bootstrap
* Secure Data Storage with MySQL
* User-Friendly Dashboard
* Fast and Efficient Record Management

## Installation and Setup

### Prerequisites

* WAMP Server
* PHP
* MySQL
* Web Browser

### Steps

1. Clone the repository:

```bash
git clone https://github.com/your-username/Gym-Management-System.git
```

2. Copy the project folder into the WAMP `www` directory.

3. Open WAMP Server and start all services.

4. Open phpMyAdmin and create a new database.

5. Import the provided SQL file into the database.

6. Configure database credentials in the project if required.

7. Open your browser and navigate to:

```text
http://localhost/Gym-Management-System
```
