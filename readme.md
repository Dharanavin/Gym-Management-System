Gym Management System

A web-based Gym Management System built using PHP, MySQL, Bootstrap, HTML, CSS, and JavaScript. The system helps gym administrators manage member records efficiently through a simple and responsive interface.

Features

- Member Registration and Management
- Add, Update, Delete, and View Member Records
- Membership Management
- Responsive User Interface using Bootstrap
- MySQL Database Integration
- Easy Record Tracking and Management

Technologies Used

Frontend

- HTML5
- CSS3
- Bootstrap
- JavaScript

Backend

- PHP

Database

- MySQL

Installation

Prerequisites

- WAMP Server
- PHP
- MySQL

Steps to Run

1. Clone the repository

git clone https://github.com/your-username/Gym-Management-System.git

2. Copy the project folder into the WAMP `www` directory.

3. Start Apache and MySQL from WAMP Server.

4. Create a database in phpMyAdmin.

5. Import the SQL file into the database.

6. Open your browser and run:

http://localhost/Gym-Management-System
